# Quick Reply — Feature Specification

## Purpose
Quick Reply generates fast, ready-to-use responses for messages, emails, comments, and DMs.

## Outputs
- 3 short quick replies
- 1 professional reply
- 1 friendly reply

## Inputs
- Received message
- Tone selection
- Length preference

## No-Code MVP Version
ChatGPT produces several reply options. The user selects one and uses it directly.

## User Flow
1. User enters the received message
2. System generates 3–5 ready-to-send replies
3. User selects the preferred reply
